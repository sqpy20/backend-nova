API Documentation

Student Routes

Student Registration

POST /api/student/register

Request:

Content-Type: application/json or application/x-www-form-urlencoded

{ "first_name": "John", "last_name": "Doe", "username": "johndoe", "email": "john.doe@example.com", "password": "securepassword" }

Response:

{ "message": "User registered successfully", "user": { "id": "user-id", "first_name": "John", "last_name": "Doe", "username": "johndoe", "email": "john.doe@example.com", "access_token": "access-token" } }

Student Login

POST /api/student/login

Request:

Content-Type: application/json

{ "email": "john.doe@example.com", "password": "securepassword" }

Response:

{ "access_token": "access-token", "user": { "first_name": "John", "last_name": "Doe", "username": "johndoe", "email": "john.doe@example.com" } }

Submit Student Data

POST /api/student/data

Request:

Authorization: Bearer access-token

{ "course_name": "Mathematics", "course_code": "MATH101", "course_unit": 3, "test_score": 80, "exam_score": 90 }

Response:

{ "message": "Student data submitted successfully" }

Fetch All Student Data

GET /api/student/datas

Request:

Authorization: Bearer access-token

Response:

[ { "course_name": "Mathematics", "course_code": "MATH101", "course_unit": 3, "test_score": 80, "exam_score": 90, "student_id": "student-id", "id": "data-id" } ]

Create Student Prediction

POST /api/student/create/prediction

Request:

Authorization: Bearer access-token

{ "course_name": "Mathematics" }

Response:

{ "message": "Prediction made successfully", "access_token": "access-token", "stored_prediction": { "course_name": "Mathematics", "decision_tree_pred_class": 1, "decision_tree_pred_prob": [0.3, 0.7], "id": "prediction-id", "linear_regression_pred": 85.5, "risk_factor": "low", "student_data_id": "data-id" } }

Get Student Predictions

GET /api/student/predictions

Request:

Authorization: Bearer access-token

Response:

{ "user_id": "user-id", "predictions": [ { "course_name": "Mathematics", "decision_tree_pred_class": 1, "decision_tree_pred_prob": [0.3, 0.7], "id": "prediction-id", "linear_regression_pred": 85.5, "risk_factor": "low", "student_data_id": "data-id" } ] }

Get Specific Student Prediction for a Specific Course

GET /api/student/predictions/<string:course_name>

Request:

Authorization: Bearer access-token

Response:

{ "user_id": "user-id", "predictions": [ { "course_name": "Mathematics", "decision_tree_pred_class": 1, "decision_tree_pred_prob": [0.3, 0.7], "id": "prediction-id", "linear_regression_pred": 85.5, "risk_factor": "low", "student_data_id": "data-id" } ] }

Admin Routes

Admin Registration

POST /api/admin/register

Request:

Content-Type: application/json or application/x-www-form-urlencoded

{ "admin_name": "Jane", "username": "adminjane", "email": "jane.admin@example.com", "password": "securepassword" }

Response:

{ "message": "User registered successfully", "user": { "id": "user-id", "admin_name": "Jane", "username": "adminjane", "email": "jane.admin@example.com", "access_token": "access-token" } }

Admin Login

POST /api/admin/login

Request:

Content-Type: application/json

{ "email": "jane.admin@example.com", "password": "securepassword" }

Response:

{ "access_token": "access-token", "user": { "admin_name": "Jane", "username": "adminjane", "email": "jane.admin@example.com" } }

Get All Students

GET /api/admin/students

Request:

Authorization: Bearer access-token

Response:

[ { "id": "student-id", "first_name": "John", "last_name": "Doe", "username": "johndoe", "email": "john.doe@example.com" } ]

Get Student by ID

GET /api/admin/student/<string:id>

Request:

Authorization: Bearer access-token

Response:

{ "student_data": { "id": "student-id", "first_name": "John", "last_name": "Doe", "username": "johndoe", "email": "john.doe@example.com" }, "data": [ { "course_name": "Mathematics", "course_code": "MATH101", "course_unit": 3, "test_score": 80, "exam_score": 90, "student_id": "student-id", "id": "data-id" } ] }
